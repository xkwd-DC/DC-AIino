## 三、项目内容（500 字以内）

围绕粮食风险研究**"数据维度单一、模型可解释性不足、研究成果难以落地"**三大痛点，本项目以**"多模态数据驱动 + 可解释 AI 双重验证 + 可视化系统落地"**为主线。

### 主要研究内容

1. **多模态数据集构建**
   - 整合 2011—2023 年 31 省份省级统计数据
   - MODIS 遥感影像（NDVI、地表温度）
   - 气象站点逐日观测
   - GIS 空间矢量数据
   - 构建覆盖致灾、气候与韧性因子的多模态数据集

2. **双模型核心因子识别**
   - **XGBoost-SHAP**：横截面识别核心因子并量化贡献度与交互效应
   - **Attention-LSTM**：注意力机制 + 时序视角独立验证稳健性

3. **可视化决策系统开发**
   - 现代 Web 技术
   - 四大模块：风险时空地图、SHAP 归因看板、参数情景模拟、韧性路径推荐

4. **韧性提升路径凝练**
   - 灾害防控、设施建设、政策保障三维度
   - 输出 31 省差异化韧性提升路径

## 五、预期效果

### （1）数据与模型

- 全国 31 省份 2011—2023 标准化多模态数据集（**约 400 条**）
- 时空对齐准确率 **≥95%**
- XGBoost-SHAP 与 Attention-LSTM 双引擎协同验证模型
- 测试集 **R² ≥ 0.62、RMSE ≤ 0.0055**

### （2）性能优化

- 模型轻量化封装与 API 部署
- 单次推理响应时间 **≤2 秒**

### （3）系统实现

- 集成"风险时空地图、SHAP 归因看板、参数情景模拟、韧性路径推荐"四大模块
