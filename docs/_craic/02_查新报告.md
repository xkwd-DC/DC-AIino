| **二．查新项目的创新要点** 1. 在数据层面，首次构建覆盖"省级面板统计 + MODIS卫星遥感 + 气象站点逐日观测 + GIS空间矢量"四类异构数据源、覆盖全国31个省份2011—2023年的多模态融合数据集，从"宏观经济统计→中观气象监测→微观地表特征"三粒度并行刻画粮食生产风险。 2. 在方法层面，提出"XGBoost-SHAP非线性归因 + Attention-LSTM时序验证"双引擎协同验证架构：基于XGBoost梯度提升树构建非线性风险拟合模型；引入SHAP（Shapley值）框架实现全局贡献度、特征交互、个体决策三维度可解释性归因；引入注意力机制升级LSTM时序模型，自适应聚焦极端气候敏感时间节点；通过两类原理完全不同的模型独立建模、交叉印证，实现对核心风险因子识别结论稳健性的双重验证。 3. 在应用层面，将算法成果工程化落地，开发集"风险时空地图、SHAP归因看板、参数情景模拟、韧性路径推荐"四大模块为一体的Web端可视化决策支持系统，构建"数据—模型—应用"全流程闭环。 4. 在风险识别结果方面，通过SHAP交互依赖图深度挖掘灌溉率与气温等因子间的非线性协同效应，揭示传统单因子分析难以发现的复杂耦合机理，并据此提出差异化的区域韧性提升路径。 |
| --- |

| **三．查新点** 1. 在多模态数据融合方面：构建覆盖"统计 + 遥感 + 气象 + GIS"四模态、全国31省份、2011—2023年的标准化多模态粮食生产风险数据集，并实现年度时间口径与省级空间口径上的多源异构数据对齐。 2. 在双引擎协同验证机制方面：将XGBoost-SHAP可解释机器学习与Attention-LSTM注意力增强深度学习两类异构模型应用于同一组多模态数据并相互独立验证，对核心因子影响力排名及作用方向进行交叉检验，破解传统单一模型"黑箱化"与稳健性不足的双重困境。 3. 在可解释性归因与交互效应分析方面：基于SHAP框架从全局重要性、个体决策、特征交互三个维度系统性解构粮食生产风险因子，通过SHAP交互依赖图挖掘气候因子与韧性因子之间的非线性协同效应。 4. 在可视化决策支持系统工程化落地方面：将研究成果从"论文产出"升级为"决策工具"，开发集风险时空地图、SHAP归因看板、参数情景模拟、韧性路径推荐于一体的Web端交互式可视化决策平台。 |

| **四．文献检索范围及检索策略** 1. 文献检索范围 时间范围：检索近5年（2021—2026年）的文献，确保信息具有时效性和前沿性。 数据库选择：使用中国知网（CNKI）、Web of Science、IEEE Xplore、Scopus、谷歌学术、ScienceDirect等权威数据库。 文献类型：包括期刊论文、会议论文、专利、学位论文、行业研究报告等，确保覆盖多角度研究成果。 研究主题：聚焦于极端气候与粮食生产风险、XGBoost-SHAP可解释机器学习、Attention-LSTM注意力增强深度学习、多模态数据融合、农业风险可视化决策系统等关键词。 2. 检索策略 关键词组合：使用中文检索词"极端气候 + 粮食生产风险"、"XGBoost + SHAP"、"LSTM + 注意力机制"、"多模态数据融合 + 农业"、"农业韧性 + 可视化系统"等，英文检索词"Extreme climate AND food production risk"、"XGBoost AND SHAP AND agriculture"、"Attention-LSTM AND time series"、"Multimodal data fusion AND crop"、"Agricultural resilience AND visualization system"等核心关键词进行组合检索。 高级检索：通过布尔逻辑（AND、OR、NOT）和通配符（*、?）优化检索表达式，并按学科领域（农业科学、计算机科学、地球科学）进行限定。 筛选标准：优先选择SCI/EI检索的高被引文献、CSCD核心期刊论文，以及与本项目直接相关的研究成果，剔除报道性、综述性文献中重复信息。 |

| **五．检索结果** 通过上述文献检索范围及检索策略，共命中相关文献约260余篇，对其中高相关性文献逐篇分析后，发现已有研究主要集中于以下方面，与本项目存在明显差异： 1. **关于极端气候与粮食生产风险研究**：国内外现有研究多采用线性计量模型或单一机器学习方法对单一气候因子（降水、温度、SPEI）影响进行评估，研究尺度多停留在省域或区域层面，缺乏对全国31省份、多极端事件耦合作用的系统性识别。本项目首次将分析尺度扩展到全国省域层面并构建多极端事件耦合识别框架，未发现相同报道。 2. **关于XGBoost-SHAP方法应用**：现有应用多集中于金融、医疗等领域，在农业风险识别方面虽有零星探索（如曹克苗（2025）针对农业风险评估的XGBoost-SHAP应用），但均基于单一统计数据来源，缺乏与遥感、GIS等多模态数据源的融合；同时SHAP分析多停留在全局重要性排序层面，未系统性挖掘特征间的非线性交互效应。本项目对SHAP的全局、局部、交互三维度系统性应用属于方法层创新。 3. **关于LSTM及注意力机制在农业时序预测中的应用**：黄鑫茜（2025）基于LSTM开展黄淮冬小麦春季低温灾害预测，李嘉乐（2024）基于深度学习开展大豆遥感估产研究，但相关研究均为单一模型独立应用，未与可解释机器学习模型形成"双引擎协同验证"机制；同时注意力机制在粮食生产风险时序识别中的应用尚处起步阶段。本项目的"XGBoost-SHAP × Attention-LSTM"双引擎协同验证架构未见相同报道。 4. **关于多模态数据融合在农业领域的应用**：现有研究多在遥感与气象数据融合方面有所探索，但同时融合"省级统计 + MODIS遥感 + 气象站点 + GIS空间矢量"四类异构数据并应用于全国尺度粮食生产风险识别的研究未见报道。 5. **关于农业风险可视化决策系统**：国内外已有部分农业监测可视化平台（如美国USDA、中国农业农村部部分项目），但多聚焦于产量监测或灾情速报，缺乏集"多模态识别 + 可解释AI归因 + 情景模拟 + 韧性路径推荐"四位一体的决策支持系统。本项目所提系统在功能模块与算法集成层面具有显著新颖性。 综合以上分析，本项目所提出的多模态数据融合体系、双引擎协同验证机制及可视化决策支持系统的整体集成方案，在所检数据库及时间范围内未发现与之完全相同或高度相似的研究报道。 |

| **六．查新结论** 1. **创新性** 本项目通过实验验证了多模态融合框架在全国尺度粮食生产风险识别中的可行性，为极端气候与粮食安全交叉研究提供了新视角；构建的XGBoost-SHAP × Attention-LSTM双引擎协同验证架构，为可解释AI与时序深度学习模型的协同应用提供了新方法；开发的Web端可视化决策支持系统，填补了国内同类研究中算法成果与可视化决策工具脱节的空白。 2. **技术先进性** 多模态数据融合方法为农业风险研究提供了可复用的数据基础与处理范式，具有较强推广价值。 在模型性能上，XGBoost模型在原始Y上R²达到0.5647（务实路线 10 种子均值实测上限）、RMSE为0.0051；Attention-LSTM验证模型R²与XGBoost接近，RMSE为0.0046。模型识别精度与稳健性均符合本项目"以韧性路径与可解释性为主要卖点"的务实路线，且核心因子识别结论在两类异构模型下保持高度一致。 在系统应用上，所开发的可视化决策支持平台支持省域风险时空地图展示、SHAP归因瀑布图分析、参数情景反事实模拟、雷达图韧性对比等交互功能，具备良好的用户体验与工程化落地能力。 3. **应用前景** 项目成果可为政府农业管理部门、农业保险机构、农业科研机构提供差异化、可解释、可操作的决策支持，对落实国家粮食安全战略、推动可解释机器学习方法与多模态融合技术在农业领域的进一步推广具有重要的理论价值与实践意义。 综上所述，本项目在数据融合方案、双引擎协同验证机制、可视化决策系统构建等方面均具有创新性和技术先进性，符合中国机器人及人工智能创新大赛的参赛要求。 |
