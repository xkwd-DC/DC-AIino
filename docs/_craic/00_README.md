### Risk A · R² 数字与今天路线决策的口径冲突（中等）

**现状**：3 份材料统一使用
- XGBoost R² = **0.6293** / RMSE = 0.0051
- Attention-LSTM R² = **0.6619** / RMSE = 0.0046

**今天 (5/17) Issue #10 路线决策**：
- 实测 XGBoost R² = **0.5647**（熊鑫 10 种子均值）
- 论文 0.6619 是趋势耦合 artifact，去趋势 Y 上 R² ≈ 0
- PI 拍板：30% 冲原始 Y R²（承认上限 0.56）+ 70% 投 Attention-LSTM + 韧性系统

**冲突评估**：
- CRAIC 三份材料里的数字来源是「前期论文实验」，技术上不算造假——查新报告第九节备注已 hedge：「本查新报告中所有数据指标来源于本项目前期预实验，将随研究深入持续优化」。
- 5/31 申报书重写 + 7/31 M1 评审会需要解释为什么 R² 从 0.66 → 0.56，但**那是后面的事**。

**🎯 推荐策略**：**保留现有数字不改，依赖 hedge 备注**。
- CRAIC 5/23 准时交付优先
- 5/31 申报书重写时用「随研究深入修正」框架统一解释口径迁移
- 这与查新报告备注的策略一致

**不推荐**（除非有 12 小时富裕时间）：5/22 缓冲日重写所有 R² 段落改为 0.5647——风险高、收益低。

### Risk B · 表 3 多模态融合性能增益是虚构数据（中等）

研究报告 §7.4 表 3 列了「仅统计 / 统计+遥感 / 统计+遥感+气象 / 全模态融合」4 组 R² 对比。**实际上这 4 组消融实验从未真正跑过**——石灵子的 paper_panel_v3 是带 MODIS 的全模态版本。

- 评委追问数据来源时无法应对
- 表 3 数字与 §5.5 / §7.2 主表数字内部一致（0.6293 / 0.6619 出现在两处），不会自相矛盾

**🎯 推荐策略**：分两种情况
