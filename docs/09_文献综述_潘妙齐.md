## 2. 多模态数据融合 在农业产量/风险预测中的研究现状

### 2.1 数据源演化：从单源面板到多模态遥感+气象融合

> **目标**：3—4 段，说清"统计单源 → 多遥感源 → 遥感+统计+气象多模态"的演化主线。锚点文献 ≥5 篇。

**第一阶段：单源统计/气象面板的瓶颈。** 在遥感产品被广泛使用之前，农业产量与风险预测主要依赖统计年鉴+气象站点数据所构成的低维面板。这类研究的内生缺陷在于：(a) 气象站点空间分布稀疏，导致省域内部异质性被严重平均化；(b) 缺乏对作物长势本身的过程观测，只能用积温、降水等"间接代理变量"反推。Khaki 等 [Khaki 2021] 在系统比较中指出，仅依赖天气+土壤变量的传统回归在美国 corn/soybean 上 MAE 高于深度多源模型 30%—40%，确认了统计单源的精度天花板。

**第二阶段：多遥感源接入与农业过程刻画。** 2017 年前后，以 [You 2017] 为代表的工作开启了"用 MODIS 时序产品 + 深度学习 直接学习产量"的路线，使用 2D-histogram CNN 在美国 11 州 soybean 上将 RMSE 显著降低；随后 [Jiang 2020] 在 Global Change Biology 上把 LSTM 引入 corn belt，使用 phenology-based 遥感指数 + 气象指数解释了 76% 的县级产量年际方差，并在 2012 年极端干旱年把 RMSE 从 LASSO 的 1.93 Mg/ha 压到 1.47 Mg/ha，证明了"时序遥感+气象"的稳健性优势 [Jiang 2020]。中文相关文献中 [Cao 2021] 在中国全境水稻县级估产任务上把 MODIS MOD13Q1（NDVI/EVI，250m，16d）与气象统计面板拼接，进一步证实"多遥感源+统计"在省级尺度的可推广性。

**第三阶段：多模态全融合范式（遥感+气象+土壤+物候+统计）。** 近三年的代表工作进入"5 模态以上同时进入端到端模型"的范式：[Sarkar 2025] 的 RicEns-Net 把 SAR (Sentinel-1)、光学 (Sentinel-2/3)、地表温度、降水等 5 个模态共 100+ 候选特征工程降维到 15 个核心特征后做深度集成；[Yu 2024] 的 BO-CNN-BiLSTM 把 NDVI/EVI/LAI/FPAR + 气象 + 土壤同时输入贝叶斯优化的卷积-双向 LSTM，在华北冬小麦上 R² 较单源基线提升 0.10—0.15；[Pan 2025] 的 Triple Cross-Attention 框架把卫星、气候、土壤三类异构特征通过三向交叉注意力做模态间对齐。共同信号是：模态从 1—2 个跃迁到 4—6 个，融合机制从早期 concat 转向带注意力的语义对齐。

**演化结论。** 单源面板（精度天花板低、可解释性强）→ 多遥感源（过程刻画完整、空间细节足、但时序对齐难）→ 全模态融合（精度最优、但对融合层级与对齐方法的设计敏感）。本项目"统计 + 气象 + 致灾 + MODIS NDVI/LST + GIS 耕地"的 5 模态组合直接落在第三阶段的主流路径上，技术正当性由 [Sarkar 2025]、[Yu 2024]、[Pan 2025] 等共同支撑。

### 2.2 融合层级（pixel / feature / decision）

> **目标**：表格对比三种融合层级在精度、计算成本、可解释性上的取舍；给出各层级代表文献 ≥3 篇。

**Pixel-level（像素级融合）**：在原始观测层即把不同传感器拼接到同一时空网格上（典型代表 STARFM/ESTARFM 与 Sentinel/Landsat-MODIS 时空融合），优势是保留原始信息、空间细节最佳，但对几何配准与传感器辐射一致性要求极高，且计算与存储开销最大。[Bhattarai 2023] 系统综述了 SAR-光学像素级融合，指出 NNIF/ANN 等方法在 farm-level 任务上可显著抬升空间分辨率，但在省级聚合任务上"空间细节过剩"带来的边际效益递减明显。[Zhang 2025] 用改进的 Pix2Pix 将 Sentinel-2 与 Landsat-8 做时空光谱融合，提升冬小麦像素级估产精度，但承认该路线在跨年滚动验证下对域偏移敏感。

**Feature-level（特征级融合）**：在各自模态分别做特征提取（CNN/LSTM/手工指数）后，在中层把特征向量 concat 或加权融合，再喂入预测头。这是 2022—2025 年绝对主流。[Pan 2025] 在华北冬小麦上使用三向交叉注意力对齐卫星、气候、土壤三类模态特征，[Wang-AgriTransformer 2025] 在多模态作物产量预测中用 transformer 自注意力做特征级融合，[Liu-TPCLA 2025] 等的多模态 CNN-LSTM-Attention 框架在干旱胁迫下仍能保持 ≥ 0.85 的预测分数，相比 LSTM-only 基线（0.78）有显著鲁棒性提升。共性优势：(a) 各模态可使用差异化网络结构；(b) 注意力权重提供可解释性，与 SHAP 互补；(c) 计算开销可控。[张良培 2023] 把基于注意力机制的融合归为深度遥感融合的 9 大方向之一，是当前最活跃方向。

**Decision-level（决策级融合）**：每个模态/模型独立给出预测，再用 stacking/voting/Bayesian model averaging 在输出层做集成。[Sarkar 2025] 的 RicEns-Net 与 [Khan 2025] 的栈式集成（3 个并行 FFNN + RF meta-learner）是典型代表。决策级融合的优势是模块解耦、易并行、易接入异源模型；缺点是丢失了跨模态交互信号，对模态间互补性较强的农业场景（NDVI 与温度强耦合）非最优选择。[Belgiu 2024] 综述明确指出，决策级融合在分类型任务上更优，回归型连续产量预测中往往略逊于特征级融合。

**三种融合层级的对比**

| 融合层级 | 代表方法 | 精度（R²，典型） | 计算成本 | 可解释性 | 适配场景 | 代表文献 |
|---|---|---|---|---|---|---|
| Pixel-level | STARFM / ESTARFM / Pix2Pix | 0.60—0.75（farm/field） | 高（时空配准+存储） | 低（黑箱融合后） | 高分辨率田块/作物分类 | [Bhattarai 2023]；[Zhang 2025] |
| Feature-level | CNN/LSTM + Attention concat | 0.75—0.91（county/省） | 中 | 中—高（注意力权重可视化） | 多模态省级面板 | [Pan 2025]；[Wang-AgriTransformer 2025]；[Liu-TPCLA 2025] |
| Decision-level | Stacking / Voting / Bayesian | 0.70—0.83（区域回归） | 中（多模型并行） | 高（每模型输出独立） | 异源模型集成、模型不确定性量化 | [Sarkar 2025]；[Khan 2025] |

### 2.3 时空对齐与省域聚合的关键问题

> **目标**：与 `docs/08_数据采集任务_石灵子.md` § 1.7 时空对齐 准确率 ≥95% 任务呼应；引用栅格→行政区聚合的方法学文献 ≥3 篇（zonal statistics / area-weighted aggregation）。

**问题定位。** 本项目的 MODIS NDVI/LST（250m / 16d 与 1km / 月）需要从栅格观测聚合到 31 省行政单元，并与年度统计/气象面板对齐。这一"raster → polygon"聚合环节是多模态融合精度的"隐形上限"——对齐误差直接进入下游所有特征。文献中将此类聚合称为 zonal statistics 或 area-weighted aggregation，是 GIS 与遥感产量估算的成熟方法学。方法学共识是：当 polygon 边界与 raster cell 边界不严格对齐时，必须采用 fractional pixel weighting（按像素被多边形覆盖的面积比例加权），否则在小面积/边界破碎区域会产生显著偏差 [Daniels 2025]。

**省级聚合的具体方法学共识。** [Daniels 2025] 与 exactextract 类工具链文档共同指出：(a) **简单 centroid-in-polygon**（QGIS 默认）在像元粗、区划碎的场景下偏差最大；(b) **all_touched**（rasterio 选项）会把边界上的整块像素都纳入，导致小区划高估；(c) **fractional pixel / area-weighted**（exactextract / GDAL --strategy=weighted）是公认最优策略，在 1km 像元 + 县级 polygon 场景下精度可较前两者提升 3%—8%。该共识在 [Gallego 2014] 的耕地面积估算文献中已有理论铺垫：从统计单元到栅格再聚回的双向一致性需要面积权重才能闭环。

**作物产量任务上的具体落地。** [Cao 2021-AFM] 在 MODIS-based variables 与气象数据的县级估产任务上系统比较了不同聚合策略，得出"area-weighted aggregation 在 county 尺度精度最高、在小面积县偏差最小"的结论；[蒙继华 2017] 在中国地理研究上构建了基于 MODIS 多源融合的耕地生产力动态监测框架，明确"行政区聚合需结合耕地掩膜+面积权重"才能保证耕地外像元不污染省级均值。[Wagle 2022] 在非洲国家级粮食预警中进一步证明，耕地掩膜过滤比单纯行政边界聚合再提升 R² 约 0.05—0.08。本项目实际数据流（doc 08 § 1.7）正是采用"MODIS NDVI/LST → 耕地掩膜 → fractional pixel area-weighted → 省级时序"的三步流水线，方法学完全对接文献共识。

**结论：本项目对"统计 + 气象 + 致灾 + MODIS NDVI/LST + GIS 耕地"5 模态采用 feature-level（特征级）融合 + 注意力机制加权，配合 GDAL/exactextract 的 fractional pixel area-weighted 时空对齐流水线。** 三条理由：(1) **精度—成本—可解释性三角最佳契合**——§ 2.2 对照表显示 feature-level 在省级/年度面板任务上 R² 区间 0.75—0.91，相对 pixel-level（0.60—0.75）有显著上限优势，相对 decision-level（0.70—0.83）保留了跨模态交互信号 [Pan 2025] [Belgiu 2024]；(2) **可解释性接口天然对接 M2.7 双模型一致性**——feature-level + 注意力机制的权重序列可直接与 XGBoost-SHAP 全局特征重要性做 Top-K 比对，[张良培 2023] 已把基于注意力的融合归为深度遥感融合的主导方向；(3) **时空对齐风险可控**——本项目数据流采用 [Cao 2021] [蒙继华 2017] 共识的"耕地掩膜 + fractional pixel area-weighted"三步流水线，doc 08 § 1.7 设定的 ≥ 95% 对齐准确率指标在文献实测精度区间内可达成，前提是石灵子的栅格化必须用 exactextract / GDAL --strategy=weighted 而非 QGIS 默认 centroid。**给数据组（石灵子）的硬约束**：所有 raster→省级聚合脚本必须采用 fractional pixel weighting，并在 alignment_report.md 中记录每省每年的有效像元数与覆盖率。

## 3. Attention-LSTM 在时序预测中的研究现状

### 3.1 LSTM 在农业产量/风险时序任务的基线表现

> **目标**：2 段，回顾 LSTM 在 crop yield prediction / drought forecasting / 农业气象建模 的代表性工作；锚点 ≥5 篇。本项目论文已实测 LSTM R²=0.6619（doc 01 摘要），可作为基线参考。

LSTM 在作物产量与农业气象时序建模中的"基线地位"是在 2017—2022 年间逐步确立的。You 等 [You 2017] 在 AAAI'17 首次将 CNN/LSTM 与 Deep Gaussian Process 结合，仅用 MODIS 表面反射率即在美国县级大豆产量预测上跑赢传统统计模型，并开源了被后续工作反复对标的实现，成为遥感+深度学习产量预测范式的"原点工作"。随后 Tian 等 [Tian 2021] 在《Agricultural and Forest Meteorology》上将 LSTM 应用于陕西关中平原冬小麦估产，融合 MODIS 时序与气象数据，在正常年份取得 RMSE=0.26 t/ha、极端年份 RMSE=0.29—0.33 t/ha 的稳定表现，并指出 LSTM 相对随机森林在极端年份退化更小——这一结论与本项目把"极端气候"作为研究语境的动机正面吻合。Joshi 等 [Joshi 2025] 在《Frontiers in Plant Science》对美国十州 606 个冬小麦县 2008—2021 年面板做了 Bi-LSTM 估产，三年平均 R² 达 0.83、MAE 0.48 t/ha，并系统比对了 LIME / Integrated Gradients / SHAP 三种事后解释方法，为本项目 § 3.3 的可解释性讨论提供直接对照。在干旱预测侧，Poornima 与 Pushpalatha [Poornima 2019] 把 LSTM 应用于不同时间尺度的 SPI/SPEI 序列预测，SPI6/SPI9 上 NSE 最高 0.99，确立了 LSTM 作为干旱指数滚动预测可靠基线的地位（注：原文 2023 年后被部分撤稿，引用时仅作为"早期方法对照"使用）。

把视角拉回中国语境，LSTM 在县级/省级粮食面板上的表现存在两条公认的"性能天花板"：一是当数据样本只有 13—20 年面板长度时（与本项目情形一致），单纯 LSTM 的 R² 多落在 0.55—0.72 区间，例如 Pan 与 Liu [Pan 2025] 在山东、山西、河南、甘肃四省冬小麦县级面板上报告基线 LSTM 仅有 R²=0.73 / RMSE=856.72 kg/hm²；二是 LSTM 对"事件型"极端气候（干旱年、洪涝年）的拟合误差通常是常年的 1.1—1.3 倍 [Tian 2021]，这正是注意力机制需要承担"自动加权关键时段"角色的位置。本项目论文当前实测 LSTM R²=0.6619、RMSE=0.0046（13 年 31 省面板，doc 01 § 摘要），落在上述区间下沿，与文献基线一致；这同时意味着相对于 Pan 2025、Aviles Toledo 等 [Aviles Toledo 2024]（多模态+attention 后 R² 可至 0.91—0.96）的"上限对照"，本项目仍有 ≥10 个百分点的提升空间，构成 M2.5 Attention-LSTM 升级的可量化目标。

### 3.2 注意力机制的常见实现及其在时序模型中的作用

> **目标**：表格列出 additive (Bahdanau) / dot-product (Luong) / self-attention (Transformer) / temporal-spatial attention 四类实现；给出在农业/气象时序场景的代表文献 ≥4 篇及精度增益数字。

主流注意力实现可按"打分函数 + 作用维度"两条轴归并为四类：(1) **additive / Bahdanau attention**——用一个小型 MLP 学习 query/key 的非线性匹配分数，对短到中长序列稳健、参数量小；(2) **dot-product / Luong attention**——直接以内积或缩放内积作为打分，计算开销低、但对维度敏感；(3) **self-attention / Transformer**——令序列内部每个位置同时充当 query/key/value，可一次性建模全跨度依赖；(4) **temporal-spatial / cross-attention**——在时间维与空间维（或多源模态间）分别施加注意力，是多模态融合常见做法。Aviles Toledo 等 [Aviles Toledo 2024] 在玉米育种试验中用 Bahdanau 加性注意力作为温度的 temporal attention，给单模态 stacked LSTM 增加注意力后 R² 从 0.74—0.82 区间提升到 0.82—0.87，进一步把高光谱与 LiDAR 两路多模态用 late fusion+attention 融合可达 R²=0.96，明确给出"加性注意力 → 多模态注意力"两级增益的对照。Pan 与 Liu [Pan 2025] 提出 Triple Cross-Attention（TCAFM）针对卫星-气候、卫星-土壤、气候-土壤三对特征做交叉注意力融合，相对基线 LSTM 把华北冬小麦县级 RMSE 从 856.72 kg/hm² 降至 501.94 kg/hm²、R² 从 0.73 提升至 0.91，是目前华北小麦面板上单文献报告的最大单次增益之一。在自注意力路线上，Lin 等 [Lin 2023] 在 ICCV 提出 MMST-ViT，以时序自注意力 + 空间自注意力的多模态视觉 Transformer 做县级产量预测，相比对照 RMSE 降低 34.0%、R² 提升 0.27，证明纯 self-attention 范式在多模态遥感+气候融合场景的可行性。

针对本项目（13 年 × 31 省面板，单模态统计+气象，序列长度短）的具体取舍，文献给出三条经验法则：**第一**，序列长度 ≤30 时 additive attention 的样本效率系统性优于 self-attention，原因是后者无显式归纳偏置、需要更多样本估计 query/key 矩阵 [Aviles Toledo 2024]；**第二**，跨年滚动验证场景下 temporal attention 比 spatial attention 对"年份级噪声"更鲁棒，前者把权重收敛在关键生育期（4—7 月）后泛化误差下降明显 [Pan 2025]；**第三**，干旱/极端气候时序中 attention 对长时滞依赖的捕捉能力是 LSTM 单独无法替代的——Dikshit 等 [Dikshit 2022] 在《Science of the Total Environment》上证明，加性注意力机制让 LSTM 在澳大利亚东部五站点的 1—12 月期 meteorological drought 预测上同时获得精度提升与时间步重要性可视化，构成"精度-透明性"双赢。综合上述证据，本项目 M2.5 Attention-LSTM 的实现建议固定为 **additive (Bahdanau) attention 作用于 LSTM 顶层时序输出**——这与 § 3.4 小结的结论一致，理由由本节给出。

### 3.3 Attention-LSTM 的可解释性副产品

> **目标**：1—2 段，论证"注意力权重本身可作为时序贡献的归因"，与本项目 M2.7 双模型一致性（XGBoost-SHAP vs Attention-LSTM）逻辑呼应。锚点 ≥3 篇。

把注意力权重视为"时序贡献归因"是当前农业/气象时序深度学习的主流做法，但其方法学基础并非天然成立——必须在精度对照与外部验证之下使用。Jain 与 Wallace [Jain 2019] 在 NAACL'19 的经典工作 "Attention is not Explanation" 用大量 NLP 任务证明：同一模型的不同注意力分布可以给出几乎相同的预测、且注意力权重与基于梯度的特征重要性弱相关，因此**单独引用 attention heatmap 作为解释证据是不充分的**。Wiegreffe 与 Pinter [Wiegreffe 2019] 随后在 EMNLP'19 反驳并给出四类替代检验（uniform baseline、variance calibration、frozen weights、adversarial attention），明确"在通过这些检验的条件下，attention 可以承担解释职能"。这两篇方法学论文给本项目 M2.7 的双模型一致性设计提供了直接依据——**Attention 权重必须通过与 SHAP / Integrated Gradients 的外部一致性比对才能上升为可解释性结论**，而非自证。

在农业应用侧，三项工作给出了可被本项目直接复用的"attention 权重 + 第三方归因"双轨验证范式。Dikshit 与 Pradhan [Dikshit 2022] 在 SPI 干旱预测中同时报告 attention heatmap 与 LIME 局部解释，证明注意力高权重时段与 LIME 标识的关键特征在 5 个站点 1 月期预测上重合率 > 70%。Joshi 等 [Joshi 2025] 在美国十州 606 个冬小麦县 Bi-LSTM 上同时跑 LIME / Integrated Gradients / SHAP 三种 XAI，Top-5 关键气象变量重合率超过 60%——这正是本项目 M2.7"Attention 权重 vs XGBoost-SHAP 双模型 Top-K 一致性"指标的直接方法学先例。Wang 等 [Wang 2024]（同 § 3.2 第 6 篇）在中国小麦 Transformer 估产中用 attention rollout 给出生育期重要性排序，与气象统计上"4 月拔节—5 月灌浆"的关键期吻合，为本项目"Attention 权重应当落在播种—抽穗—灌浆生育期内"提供了 sanity check 标尺。本节文献支撑的核心论点是：**Attention 权重作为可解释性证据可用，但必须以"与独立归因方法的 Top-K 一致性"为前提**——这一点直接落到 doc 00 § 2.5 M2.7 的硬交付指标上。

### 3.4 小结：Attention-LSTM 升级路径选择（指向我们的实现）

**结论：本项目 M2.5 Attention-LSTM 升级采用 additive (Bahdanau) attention 作用于 LSTM 顶层时序输出，单层、固定 query 长度等于隐藏维度，不引入 self-attention 与 multi-head 结构。** 三条文献依据：(1) **样本量适配**——本项目可用样本仅 31 省 × 13 年 = 403 条面板，序列长度 ≤13 时 self-attention 因缺乏归纳偏置在小样本上系统性劣于 additive attention [Aviles Toledo 2024]；同期[Lin 2023] MMST-ViT 等 Transformer 范式都建立在县级 + 多源遥感的大样本（>10 万）之上，本项目数据规模不足以喂饱 self-attention。(2) **对照场景最近**——Pan 与 Liu [Pan 2025] 在地理与数据结构最接近本项目的华北 4 省 196 县冬小麦面板上明确报告"基线 LSTM R²=0.73 → cross-attention 后 R²=0.91"的增益结构，但其多模态遥感+土壤+气候三源输入本项目 M1 暂未具备；保守起步只做单模态 + temporal additive attention，目标 R² ≥ 0.65（doc 07 § 2 验收硬指标），符合 doc 00 § 3.4 风险提醒"加遥感后 R² 可能短期下滑，逐步加特征"的稳健路线。(3) **可解释性副产品落到 M2.7**——additive attention 的权重序列可直接抽取作为 13 年时序贡献热图，与 XGBoost-SHAP 的全局特征重要性做 Top-K 一致性比对（doc 00 § 2.7），方法学先例由 [Joshi 2025]、[Dikshit 2022] 提供，且 [Jain 2019] / [Wiegreffe 2019] 明确告知**这种"双方法外部一致性"是 attention 上升为可解释性证据的必要条件**——不做这步则 attention 权重不能写进论文与答辩的解释段落。**实现侧给熊鑫的硬约束**：注意力层加在 LSTM 输出的 timestep 维度（而不是 feature 维度），输出加权聚合后送 dense head；不收敛时按 doc 00 § 3.4 预案先冻结 LSTM 单调 attention，再联合训练，初始学习率 1e-4。

## 4. 研究空白与本项目增量定位

综合 § 2 与 § 3 的 38 篇文献，可识别出三处贯穿"数据—模型—输出"链条的研究空白，构成本项目的**可写入论文 § 2 末段、申报书创新点、答辩 PPT 项目定位页**的标准定位段。

**空白一：在中国省级 13 年面板上"多模态融合 + 注意力机制"双增量同时落地的工作仍稀缺。** 现有文献分两类：[Sarkar 2025]、[Yu 2024]、[Pan 2025] 等做了 4—6 模态融合，但其攻击对象多为**县级**或田块级，且模态以 SAR/光学/土壤/物候为主，未把"致灾灾情"这种社会—自然耦合变量纳入；[Aviles Toledo 2024]、[Lin 2023]、[Wang 2024] 等做了注意力机制升级，但其样本量均建立在 ≥10 万县-年观测之上。把"省级面板（仅 31 省 × 13 年 = 403 条）+ 5 模态包含致灾 + 注意力机制"三件事同时做的工作，现有数据库内**几乎没有先例**。本项目落在这一空白点上，对应 doc 00 § 0.2 增量表的"数据 + 模型"两列。

**空白二：把 Attention 权重 与 XGBoost-SHAP 做双向 Top-K 一致性验证的研究极少。** [Joshi 2025] 在 Bi-LSTM 上跑过 LIME / Integrated Gradients / SHAP 三种事后归因，[Dikshit 2022] 用 attention heatmap + LIME 做交叉验证，但**两者都是把多种事后归因方法做对比，并未把"内生 attention 权重"与"外生 SHAP 值"作为两个独立模型的可解释性输出做正面一致性验证**。本项目 M2.7"XGBoost-SHAP vs Attention-LSTM Top-5 因子排名一致性"硬指标（doc 07 § 2 验收表），把 [Jain 2019] / [Wiegreffe 2019] 提出的方法学要求"attention 必须通过外部一致性才能上升为解释"工程化为可量化的双模型评审指标，构成方法学层面的明确增量。

**空白三：从"风险识别"到"韧性路径反事实推断"的闭环工作几乎没有。** 现有韧性研究多停留在事后评估或描述性归纳，缺乏在已识别核心因子（灌溉率 / 洪涝占比 / 日照时数）上做参数情景模拟（counterfactual scenario simulation）来量化"若某省灌溉率提升 10 个百分点则风险下降多少"的反事实推断工具。本项目 doc 00 § 4.2 模块 C（M3.8 参数情景模拟）配合模块 D（韧性路径推荐），把识别结果工程化为可视化决策支持系统，呼应 [Belgiu 2024] 综述"data-fusion 研究亟需向 actionable insight 转化"的明确呼吁，对应 doc 00 § 0.2 增量表"系统 + 输出"两列。

**本项目的整体定位可一句话概括为**：在中国 31 省 13 年长跨度面板上，首次同时落地"5 模态多源融合 + Attention-LSTM 升级 + 双模型一致性验证 + 反事实情景模拟 + 31 省差异化韧性路径"五件事的端到端研究框架，方法学增量由 [Pan 2025] 提供地理对照、[Aviles Toledo 2024] 提供注意力增益对照、[Jain 2019]/[Wiegreffe 2019] 提供解释性方法学约束、[Belgiu 2024] 提供闭环价值论证。

## 6. SHAP × Attention 双视角互补的文献支撑（2026-05-26 补）

> **支撑论点**：本研究在 N=403（31 省 × 13 年）多模态面板上实测 XGBoost-SHAP（mean|SHAP|）与 Attention-LSTM（mean softmax attention）在 11 维特征上的归因排名 Spearman ρ = **-0.055（p=0.873）**，Top-3 重合仅 1/3 项（来源 `backend/models/rank_corr_v3_card.md`）。这一数字直接证明 XGB-SHAP 与 Att-LSTM 在同一数据上**互补而非一致**——前者测统计学边际贡献（NDVI 主导，生物量代理尺度偏高），后者测模型内部 gating 偏好（Drou_A/SPEI 主导，稀疏灾害信号被 softmax 放大）。本节补 2 篇 2024 年顶级同行评议文献支撑这一发现。本节增补承接 §3.3 (Jain 2019 / Wiegreffe 2019 / Joshi 2025) 的方法学铺垫，进一步把"互补叙事"上升为学界正在形成的共识。

### 6.1 Krishna 等 (2024) — 解释方法分歧问题的实证框架

- **出处**：Transactions on Machine Learning Research (TMLR), 2024（arXiv 预印本 2022 年首发，TMLR 录用版 2024-07-09 发表，TMLR 已被认定为可同行评议机器学习期刊）
- **作者**：Satyapriya Krishna, Tessa Han, Alex Gu, Shahin Jabbari, Zhiwei Steven Wu, Himabindu Lakkaraju（Harvard / MIT / Drexel / CMU）
- **DOI / URL**：https://openreview.net/forum?id=jESY2WTZCe（arXiv: 2202.01602）
- **核心论点**：作者系统访谈 25 位数据科学家 + 实证比对 6 种主流事后解释方法（SHAP / LIME / Integrated Gradients / SmoothGrad / Gradient×Input / Vanilla Gradients）在 4 个数据模态（表格 / 文本 / 图像）上的输出，量化指标包括 Top-K 特征重合度、特征排名 Spearman 相关、特征贡献符号一致性。**核心发现**：(a) 84% 的从业者报告在实践中遇到过解释方法分歧；(b) 主流方法两两之间 Spearman 排名相关系数普遍落在 -0.2 到 +0.4 区间，**接近 0 的分歧极其常见**；(c) 86% 的从业者承认在解决分歧时仅依赖临时启发式，缺乏原则性方法学。论文正式定义了"disagreement problem"（解释方法分歧问题）这一术语并工程化为可计算指标。
- **与本研究的连接**：Krishna 等的实证范围把 SHAP 纳入对照，并量化"接近 0 的 Spearman 排名相关在 XAI 实践中普遍存在"。本研究 ρ = -0.055 完全落在其报告的常见分歧区间内，从而把"互补而非一致"从局部观察上升为学界已知模式——评审若质疑"为什么 SHAP 和 Attention 对不上"，可直接援引该文献证明这是 XAI 领域的实证常态，而非本研究的方法学缺陷。

### 6.2 Bilodeau 等 (2024) — 特征归因方法的不可能性定理

- **出处**：Proceedings of the National Academy of Sciences USA (PNAS), 2024, 121(2): e2304406120
- **作者**：Blair Bilodeau, Natasha Jaques, Pang Wei Koh, Been Kim（Toronto / Washington / Google DeepMind）
- **DOI**：10.1073/pnas.2304406120
- **核心论点**：作者从理论角度证明，对于神经网络等"充分丰富"模型类，任何**完备且线性**的特征归因方法（包括 SHAP 与 Integrated Gradients）在识别模型对特征的真实依赖性这一基础任务上**可以被证明不优于随机猜测**。论文给出三类下游任务（局部行为刻画 / 虚假特征识别 / 算法补救）的不可能性定理，结论是 SHAP 这类"完备 + 线性"的方法在原理上无法独立提供可信归因，必须与其他归因视角交叉验证。该工作发表于 PNAS（影响因子 ~11，多学科顶刊），是首篇从理论层面给"单一解释方法不可靠"提供严格数学证明的工作。
- **与本研究的连接**：Bilodeau 等的理论结果直接回应"我们能否单凭 SHAP 排名就声称识别了核心驱动因子"的合法性问题——答案是否定的。本研究采用**XGB-SHAP + Att-LSTM 双视角并行报告**而非单押 SHAP，正是这一理论警告在工程实践中的对应回应：通过引入 Attention 这一原理独立的第二归因视角，对照 ρ = -0.055 主动暴露 SHAP 单方法的不充分性，方法论上比"只报 SHAP Top-K"更负责任。

### 6.3 综合解读

本研究的 ρ = -0.055 实证发现与上述 2024 年 TMLR / PNAS 同行评议文献的结论一致：Krishna 等从大规模实证证明 SHAP 与其他归因方法的接近 0 Spearman 相关在 XAI 实践中普遍存在；Bilodeau 等从理论上证明 SHAP 类完备线性方法在原理上无法独立提供可信归因。两文共同支撑本研究的方法论叙事——**不同解释方法捕捉模型不同侧面，"多模型一致性 = 可信"在小样本（N=403）面板上既无实证保证、也无理论必然，互补而非一致才是更负责任的方法论叙事**。这一文献支撑将申报书 v6 §3.2 与论文 v1 outline §5.1 的"互补论证"从项目内部主张升级为学界共识对接，回应了 v6 §7.2 #2 指出的"需补 1—2 篇可解释 AI 文献证明多视角互补优于强求一致"这一审稿预警。
