## 2. 主交付 schema · `paper_panel_v3.parquet`

403 行（31 省 × 13 年） × 27 列，0 NaN，省名 / 省码 / 年份与论文已有面板严格一致。

> **v4 接口预告**（详见 `docs/16_v4_plan.md`）：v3 → v4 三方向扩展：
> - **8a 致灾真实化**：`flood` / `flood_a` / `drou_a` 三列从 SPEI 派生 proxy 替换为水利部公报 / 国家统计局《灾害基本情况》真实统计源，并拆出 `flood_severe_a` / `drou_severe_a`（成灾面积）/ `prec_50mm_days`（暴雨日数）等新增列
> - **8b 耕地像元加权**：MODIS NDVI / LST 省域聚合从"全省域均值"切换为"耕地像元加权均值"（CLCD 30m mask × MODIS 1km）
> - **8c 多年 CLCD**：CLCD 2022 单年套全周期 → 2011–2023 逐年快照
> - **影响 11 维契约**：v4 可能扩到 12–14 维，`feature_columns.json` 需要 PR + 后端 retrain 同步（与熊鑫协调 retrain 时机）
> - **8/31 硬截止**：v4 final 入库，论文 §2 数据章 + §4 实验章接 v4 数据

### 6.1 灾害数据来源确认（原"SPEI 派生 proxy"已作废）

- **2026-06-02 确认**：`flood` / `flood_a` / `drou_a` 三列**已是真实统计数据**，来源于王天硕总表.xlsx（国家统计局 /《中国水旱灾害公报》）。v3 生成时已接入，403/403 行与总表完全一致。
- **旧描述作废**：之前推测的 `flood = max(0, SPEI × 30)` 公式经实际验证不成立（SPEI 与洪涝占比的相关系数 = -0.07，与真实水灾面积的相关系数 = 0.28）。

### 6.2 MODIS 未做耕地像元加权（doc 08 §3.2 spec 偏离）

- **现状**：`scripts/data/05_aggregate_modis_to_province.py` v0 标注"没做耕地 mask"，省域聚合是**全省域均值**而非耕地像元加权
- **影响**：含大量非耕地（沙漠 / 高原 / 水体）的省份（新疆 / 青海 / 西藏 / 内蒙）NDVI 会被系统性低估
- **valid_pixel_ratio P50 = 0.091**（QA 过滤后保留像元 / 总像元）— 像元绝对数仍足够（NDVI valid count P50 数万级）
- **下一步**：CLCD 已下 2022 一年，要 fix 需补齐 2011-2023 逐年（COG 流式策略见 `scripts/data/README.md` 方案 A），重跑聚合后所有省 × 月份 NDVI 会变（预计山地 / 干旱省份 NDVI 显著上升）

### 6.3 CLCD 单年（暂用 2022 套全年）

- **现状**：当前仅下载 `CLCD_v01_2022_albert.tif`；其他年份的耕地像元应使用该年 CLCD
- **影响**：东北开荒 / 南方退耕的趋势信号被冲掉（用 2022 套到 2011 会高估当时耕地面积）
- **下一步**：补 2011-2023 逐年 CLCD（约 10 GB / Zenodo COG 流式），与 §6.2 一并 fix

### 6.4 `sun` 是辐射比近似（非真实日照时数）

- **公式**：`sun = 4380 × ALLSKY / CLRSKY`（NASA POWER 实际 vs 晴天辐射比 × 一年总小时数）
- **影响**：与气象台站实测日照时数偏高约 1.3×；**Z-score 标准化后等价**（论文流水线接受）
- **下一步**：M2 阶段若需绝对值，改用中国气象数据网 SURF_CLI_CHN_MUL_DAY 日照时数

### 6.5 NASA POWER 单点 ≠ 面均

- **现状**：使用省会经纬度单点 query，非省域面均
- **影响**：大省（新疆 / 内蒙 / 青海 / 西藏 / 四川）显著偏离面均；小省（北京 / 天津 / 上海）影响小
- **下一步**：M2 可换用 CHIRPS / ERA5-Land 面网格再做 zonal_stats（NASA POWER 不提供网格 API）

### 6.6 王天硕 Y 列漂移（已修复）

## 7. 鲁棒性验证（v3 数据上的诚实披露）

> 数据集说明的诚实性不仅在 schema 与口径，更在**模型在该数据上的真实泛化能力**。本节披露 v3 数据 + 11 维特征 + XGB / Att-LSTM 模型组合的鲁棒性三协议结果，**对外材料（论文 / 申报书 / 答辩 / 软著）所有相关数字必须与本节一致**。
>
> 数据源：`backend/models/strict_cv_v3_card.md` ｜ `backend/models/ndvi_ablation_v3_card.md` ｜ `backend/models/rank_corr_v3_card.md` ｜ `docs/_craic/robustness_summary_2026-05-22.md`

### 7.1 三协议交叉验证 R²

| 协议 | R² | RMSE (kg/ha) | folds | 含义 |
|---|---|---|---|---|
| **random 8:2**（论文协议） | **0.907 ± 0.038** | 312.5 | 10 seeds | 同省份同时进 train / test，模型识别省份 baseline 输出该省历史均值 |
| **leave-year-out** | **0.894** ± 0.104 | 296.4 | 13 | 同省份新年份预测 → ✅ 时序泛化可用 |
| **stratified 5-fold** | 0.909 ± 0.027 | 311.2 | 5 | row 级混合泛化（同 random） |
| **leave-province-out**（严格协议） | **中位 -16.83**（mean -176.9 方差极大） | 825 | 31 | 新省份预测 → ❌ **空间外推完全失败** |

**leave-province-out 分省统计**（最关键诚实披露）：
- R² > 0 的省份：**1 / 31**
- R² > 0.5 的省份：**0 / 31**
- 最佳省：R² = 0.173
- 最差省：R² = -4051.7
- 25-75 分位区间：[-74.7, -6.32]

**解读**（论文 §7.1 / 申报书 §7 局限段诚实表述）：

> random 8:2 协议下 R² = 0.907 等价于"31 省 baseline 记忆 + 时间信号 + 气候残差"。严格 leave-province-out 暴露 11 维输入**不编码可迁移的 climate → yield 映射**——0.907 的本质是 31 省 baseline 识别（分类式任务），而非气候驱动单产预测能力。这不是模型失败，而是 **N = 403 / 31 截面单位的数据上限**。

### 7.2 NDVI ablation（特征移除消融）

| 特征集 | R² | ΔR² |
|---|---|---|
| 11 features（含 NDVI） | 0.907 ± 0.038 | — |
| 10 features（移除 NDVI） | 0.904 ± 0.022 | **-0.003** |

**含义**：NDVI 对头条 R² 贡献 ≈ 0%。**对外叙事不可写"NDVI 是核心驱动因子"**——NDVI 是 baseline 区分因子（在多元输入签名里帮助识别"这是河南"），不是边际预测因子。

### 7.3 SHAP × Attention 双视角互补（非一致性验证）

| 统计量 | 数值 | p-value | 解读 |
|---|---|---|---|
| Spearman ρ | **-0.0545** | 0.873 | 弱 / 无单调一致性 |
| Kendall τ | +0.0182 | 1.000 | 弱 |
| Top-3 重合 | 1/3 | — | 主要分歧 |
| Top-5 重合 | 3/5 | — | — |

**Top-3 对比**：
- XGB SHAP top-3：`['NDVI', 'Prec', 'Temp']`
- Att-LSTM Attention top-3：`['Drou_A', 'SPEI', 'Temp']`

**解读**（论文 §7.5 诚实表述）：

> SHAP 与 Attention 排名 Spearman ρ = -0.055（p = 0.873），Top-3 仅 1/3 重合。**不应视为不一致性**，而是**方法论互补**：SHAP 测"边际贡献"（NDVI 主导，反映生物量代理）；Attention 测"模型内部 gating"（Drou_A / SPEI 主导，反映灾害稀疏信号需要主动 focus）。两个视角各自正确，**不应强求一致**。对外材料**不可写"SHAP × Attention 一致性验证"**——必须写"双视角互补归因"。

### 7.4 叙事红线（AGENT_BRIEFING §4 守门）

| ❌ 严禁表述 | ✅ 诚实表述 |
|---|---|
| "R²=0.91 高精度预测" | "random 8:2 协议 R²=0.91（论文协议）；严格 leave-province-out 退化至中位 -16.83，反映 N=403/31 截面数据上限" |
| "SHAP × Attention 一致性验证" | "SHAP 与 Attention 双视角互补（非一致性验证），Spearman ρ=-0.055" |
| "NDVI 是核心驱动因子" | "NDVI 是 baseline 区分因子（ΔR²=-0.003 移除后近乎无影响），非边际预测因子" |
| "climate → yield 高精度预测" | "省内风险因子相对重要性识别 + 决策支持系统应用层闭环" |
| "构建预测系统" | "构建决策支持系统：已知省份在历史输入空间内的情景模拟与韧性路径推荐" |
