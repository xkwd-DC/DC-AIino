### 0.1 已完成(2026-05-22 实测口径,远超原计划进度)

- ✅ **多模态数据集 v3**:31 省 × 2011—2023 = N=403,11 维 X(统计 + 气象 + 致灾 + MODIS NDVI/LST + GIS CLCD),`paper_panel_v3.parquet`
- ✅ **三模型互补训练落地**(2026-05-21 路线 E,10 种子 random 8:2 均值):
  - XGBoost-SHAP **R² = 0.9072 ± 0.038** / RMSE 312.5 kg/ha
  - LSTM **R² = 0.8856 ± 0.022** / RMSE 362.0 kg/ha
  - **Attention-LSTM R² = 0.8160 ± 0.050** / RMSE 456.4 kg/ha
- ✅ **三套验证协议并行报告**(国家级评审深度):
  - random 8:2(论文协议)/ leave-year-out R²≈0.89(时间外推)/ leave-province-out 中位 R²=-16.83(空间外推上限**诚实披露**)

  - XGB-SHAP Top-3 = `NDVI > Prec > Temp`(数据驱动 / 生物量主导)
  - Att-LSTM Attention Top-3 = `Drou_A > SPEI > Temp`(灾害响应 / policy lever)
  - SHAP × Attention Spearman ρ = **-0.055**(方法论独立性,见 `backend/models/rank_corr_v3_card.md`)
- ✅ **决策支持系统骨架**(Vue3 + Flask + SQLite,M01-M04 四模块 Vue 化):风险时空地图 / SHAP 归因看板 / 参数情景模拟 / 韧性路径推荐

- ✅ **部署素材就位**:域名 `grainrisk.app` 注册 + nginx.conf + certbot 脚本(PR #33,待 GCP 实例开机执行)
