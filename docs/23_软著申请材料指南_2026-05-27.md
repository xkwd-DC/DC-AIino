## §1 软著申请基本流程

### §1.1 申请通道

**通道 A · 中国版权保护中心(CCPC,官方权威)**

- 官网:`copyright.gov.cn` → 软件著作权登记 → 在线提交(自助申请)
- 申请人:潘妙齐(自然人) + 山西农业大学(法人,合作申请,见 §5.3)
- 审查周期:**普通 30-40 工作日**(即 ~1.5-2 个月);**电子加急 7 工作日(+1000 元)** — 9/30 提交后普通流程 11 月初下证,**符合结题时间窗**
- 登记费:**学生身份普通申请 ~250-300 元**(2026 年现行减免后);加急 +1000 元

**通道 B · 省级版权办替代渠道**

- 山西省版权局(太原市)— 通过省版权局接收后转 CCPC
- 时间略长(+5-7 工作日),但本省学生项目可走"省内绿色通道"加快受理
- 费用与 CCPC 一致

**通道 C · 学校科技处 / 教务处集中代办(推荐)**

- 学生项目通常每学期由学校科技处统一集中代办,**学校承担部分或全部登记费**(各校政策差异,需问邱老师 / 教务处)
- 优点:① 费用减免(可能 ~50-100 元甚至免费)② 学校已熟悉流程,出错率低 ③ 形式审查由学校先把关
- 缺点:① 排队(每学期统一 1-2 批次)② 灵活度低,9/30 deadline 必须卡好学校代办批次
- **本项目建议**:先问邱老师 + 教务处科技处,确认学校代办的批次时间窗。若学校 9 月有代办批次 → 优先走通道 C;若错过 → fallback 通道 A 自助申请保底

### §1.2 申请时间窗对齐 9/30

- **9/30 提交** → 普通审查 30-40 工作日 → **11 月初拿到电子证书,11 月底拿到纸质证书**
- 国家级结题验收时间窗:2027 年 4 月(N4 起止 = 2026.6—2027.5,实际答辩 4 月) → 软著 11 月初下证后提供至少 5 个月缓冲,**充分覆盖结题硬指标提交**
- 即使审查中途收到补正通知(常见,~30% 概率),补正后再审通常 +10-15 工作日,11 月中下旬下证仍然可控

### §1.3 项目负责人挂第一的法定要求

**国家级结题硬指标(`docs/national/19_结题验收要求.md` 第一条第(2)款)**:

> 项目负责人以排名前三作者获得 1 项发明专利、**第一作者获得 1 项实用新型或外观设计专利,或软件著作权**

软著的"第一作者"对应申请表上的**第一著作权人 / 第一作者**位置。本项目:

- **第一著作权人 = 潘妙齐**(强制,不能让位)
- 后续作者:熊鑫 + 石灵子 + 常宇璇(贡献排序),具体见 §5.3

### §3.4 模型集成(~3 页)

**重点**:

- **§3.4.1 三模型互补架构**(强制使用"互补"而非"一致")
  - **XGBoost-SHAP**:横截面非线性归因(边际贡献),输出全局 SHAP / 局部 SHAP / 交互依赖图
  - **基线 LSTM**:时序拟合参照
  - **Attention-LSTM**:模型内部 gating 偏好揭示(softmax attention 权重)
- **§3.4.2 推理服务架构**:`backend/services/inference.py`(324 行)负责模型 lazy load + 单次推理 + 批量推理;`/api/predict` POST 接口(`backend/api/predict.py`,411 行)对外暴露推理能力
- **§3.4.3 R² 三协议口径(诚实披露,强制使用)**
  - **random 8:2 R² = 0.907 ± 0.038**(论文协议,可引用)
  - **leave-year-out R² = 0.894 ± 0.10**(时序泛化可用)
  - **leave-province-out 中位 R² = -16.83**(31 省仅 1 省 R² > 0,空间外推完全失败,**诚实披露**)
  - **重点**:本系统的价值定位是"省内因子相对重要性识别 + 反事实情景模拟",**不依赖跨省外推能力**。leave-province-out R² < 0 不影响系统四模块功能输出(详见申报书 v6 §3.4 / 论文 v1 §6.3 / §7.3)
- **§3.4.4 NDVI 消融报告**:去 NDVI 后 R² 几乎不变(-0.003),证明 NDVI 头条贡献来自 SHAP 边际尺度而非模型主导信号。**严禁写"NDVI 是核心驱动因子"**
