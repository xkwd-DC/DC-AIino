# v5 粮食产量目标变量采集指南

> 对应 Issue #54「石灵子任务：产量目标变量采集」
> 优先途径：山西农大 EPS 数据平台（校内网 http://www.epsnet.com.cn/）

## 产出格式

无论从 EPS 导出还是统计局补充，最终产出为 CSV：

```csv
省,地级市,年份,粮食产量(万吨)
河南,郑州市,2011,xxx
河南,郑州市,2012,xxx
...
湖北,武汉市,2025,yyy
```

## EPS 操作步骤

1. 连接山西农大校内网（或 VPN）
2. 访问 http://www.epsnet.com.cn/
3. 进入 → **中国区域经济数据库**
4. 搜索指标：**粮食产量(万吨)**
5. 区域选择：
   - 黑龙江(13市)、河南(18市)、山东(16市)、吉林(9市)
   - 安徽(16市)、湖南(14市)、河北(11市)、四川(21市)
   - 江苏(13市)、湖北(17市)
6. 时间选择：**2011-2025**
7. 导出格式：**CSV 或 Excel**

> 如果 EPS 单个导出限制次数，可分省导出再拼接。
