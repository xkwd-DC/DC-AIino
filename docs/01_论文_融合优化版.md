## 摘 要

在全球气候变暖背景下，极端天气频发已成为威胁我国粮食安全的核心变量。本文基于 2011—2023 年我国 31 个省份的面板数据，针对传统研究在因子选择与模型可解释性方面的不足，构建了 **"XGBoost-SHAP + LSTM"** 双重验证的研究框架，对粮食单产风险进行系统建模与稳健性检验。

研究分两阶段展开：

- **第一阶段：** 利用 XGBoost 算法建立气候因子与生产要素对粮食单产风险的非线性映射关系，并引入 SHAP 解释框架，从全局贡献、特征交互及个体决策三个维度对风险因子进行解构。
- **第二阶段：** 引入长短期记忆网络（LSTM）作为深度学习验证模型，刻画风险的时序演化特征。

**实证结果：**

| 指标 | XGBoost 模型 | LSTM 验证模型 |
|------|---------------|----------------|
| 决定系数 R² | 0.6293 | **0.6619** |
| 均方根误差 RMSE | 0.0051 | **0.0046** |

**核心发现：** 灌溉率、洪涝占比与日照时数是影响粮食单产风险的核心因子，其中完善的灌溉设施能显著抵御极端干旱风险；同时各因子间存在显著的非线性交互关系。在双重验证下，核心因子的影响力排名及作用方向均保持高度一致。

**关键词：** 极端气候；粮食生产风险；XGBoost-SHAP；LSTM；农业韧性

## 一、引言

### （一）研究背景与意义

全球气候变暖导致极端天气事件频发，呈现范围广、强度高、频率上升的态势。IPCC《Climate Change 2022》指出气候变化正加剧全球农业系统的脆弱性[1]。

**典型案例：** 2023 年台风"杜苏芮"引发河北省极端降水，致 87 个县、54 万人受灾，约 400—500 万吨玉米（占全国全年产量 2%）受损，当年夏粮产量较上年减少 127 万吨。

**研究方法痛点：**
1. 现有研究多聚焦微观主体或单一灾种，缺乏多源数据融合的系统性风险识别框架；
2. 传统机器学习模型"黑盒化"，可解释性不足；
3. 难以精准识别核心风险因子、刻画因子间的非线性关联。

本文采用 **XGBoost-SHAP 可解释机器学习与 LSTM 深度学习模型双重验证**，通过 SHAP 值直观呈现风险因子的影响程度与作用方向，为保障国家粮食安全提供科学参考。

### （二）研究内容与研究路线

1. **核心因子识别**：基于 2011—2023 全国 31 省份面板数据，以粮食单产波动为风险表征，筛选关键变量。
2. **双模型构建与实证分析**：XGBoost-SHAP（非线性建模 + 归因）+ LSTM（时序验证）。
3. **韧性提升路径优化**：从灾害防控、设施建设、政策保障三维度提出韧性提升路径，并设计可视化决策支持系统构想。

### （三）创新点

1. **研究视角**：突破传统"线性假设"，整合多种极端天气因素，实现"主动预判 → 提升韧性"。
2. **方法体系**：XGBoost-SHAP 高精度预测 + 透明归因双重模型 + 预测+可解释性框架结合。
3. **应用价值**：构建可视化决策支持系统设计构想，为农业政策制定与防灾减灾提供决策支撑。

## 二、国内外研究现状及综述

### （一）国外研究现状

- **风险研究脉络**：从单一气候因子 → 多极端事件耦合风险（洪涝、干旱、高温）。
- **评估方法**：传统计量模型 → 机器学习（随机森林、XGBoost）→ 深度学习时序建模。
- **韧性研究**：抗旱品种 + 灌溉升级 + 政策协同；但多基于发达国家，难以适配发展中国家"小农"模式，且多为静态评估。

### （二）国内研究现状

- 形成"风险评估 - 影响机制 - 预警防控"研究脉络；
- 我国粮食生产气候风险呈区域分化、季节性集中；
- 机器学习应用提升预测精度但路径解析不足；
- 韧性研究起步较晚，缺乏"风险识别 - 韧性评估 - 路径优化"全链条研究。

### （三）综述与本文定位

现有研究的三方面不足：
1. 局限于单一气候因子或局部地区；
2. 多为静态韧性评估；
3. 多为单一模型应用，双重验证或多模型融合较少。

本文以 31 省份为对象，构建"风险评估 - 韧性评估 - 路径优化"全链条分析框架。

## 三、研究方法与数据来源

### （一）理论基础与变量选取

**1. 粮食生产风险量化公式**

$$R = | Y - \hat{Y} |$$

其中 $R$ 为粮食生产风险水平，$Y$ 为实际产量，$\hat{Y}$ 为模型拟合预期产量。

**2. 数据来源**

| 类型 | 来源 |

| 农业生产数据 | 《中国统计年鉴》《中国农村统计年鉴》、各省统计公报 |
| 气象监测数据 | 中国气象数据网（CMDC）省域逐日监测数据 |
| 致灾因子数据 | 历年《中国水旱灾害公报》 |

3. **标准化**：Z-score 标准化，$x' = (x - \mu) / \sigma$。

### （四）建模方法

#### 1. XGBoost 风险拟合模型

$$\hat{y}_i = \sum_{k=1}^{K} f_k(x_i), \quad f_k \in \mathcal{F}$$

**目标函数：**

$$\text{Obj} = \sum_{i=1}^{n} L(y_i, \hat{y}_i) + \sum_{k=1}^{K} \Omega(f_k)$$

**正则项：**

$$\Omega(f) = \gamma T + \frac{1}{2} \lambda \|\omega\|^2$$

#### 2. SHAP 值归因

$$\varphi_j = \sum_{S \subseteq M \setminus \{j\}} \frac{|S|! (M-|S|-1)!}{M!} \cdot [f(S \cup \{j\}) - f(S)]$$

$$\hat{y}_i = \varphi_0 + \sum_{j=1}^{M} \varphi_{ij}$$

#### 3. LSTM 门控机制

| 门 | 公式 |
|----|------|
| 遗忘门 | $f_t = \sigma(W_f \cdot [h_{t-1}, x_t] + b_f)$ |
| 输入门 | $i_t = \sigma(W_i \cdot [h_{t-1}, x_t] + b_i)$ |
| 候选状态 | $\tilde{C}_t = \tanh(W_C \cdot [h_{t-1}, x_t] + b_C)$ |
| 状态更新 | $C_t = f_t \odot C_{t-1} + i_t \odot \tilde{C}_t$ |
| 输出门 | $o_t = \sigma(W_o \cdot [h_{t-1}, x_t] + b_o)$ |
| 隐状态 | $h_t = o_t \odot \tanh(C_t)$ |
| 输出 | $\hat{y}_t = W \cdot h_t + b$ |

### （五）评价指标

$$R^2 = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2}$$

$$\text{RMSE} = \sqrt{\frac{1}{n} \sum (y_i - \hat{y}_i)^2}$$

## 四、基于 XGBoost-SHAP 的核心因子识别

### （一）拟合性能评估

- **数据划分**：训练集:测试集 = 8:2
- **超参数**：学习率 0.1，最大深度 6，迭代次数 150（Grid Search 优化）
- **测试集表现**：R² = **0.6293**，RMSE = **0.0051**

### （二）特征重要性全局排序

> **Top 3：灌溉率 > 洪涝占比 > 日照时数**

人为调节能力（灌溉）+ 气候冲击（旱涝）= 决定风险波动的核心力量。

### （三）SHAP 蜂群图核心因子解释

- **韧性因子**：灌溉率与日照时数高值（红色）集中在 0 轴左侧 → 减损作用。
- **致灾因子**：洪涝占比与降水量高值集中在右侧 → 推高风险；SPEI 蓝色点（干旱）SHAP 值为正 → 缺水破坏力强。

### （四）灌溉率 × 平均气温交互效应

- **主效应**：灌溉率提升 → SHAP 值下降 → 减损。
- **交互效应**：相同灌溉水平下，气温越高 SHAP 值越大 → 高温会部分抵消灌溉效果。
- **结论**：必须结合温度调节等综合措施。

## 五、基于 LSTM 的双重验证

### （一）模型架构

- 输入层 → 两层 LSTM 隐藏层 → 全连接输出层
- Adam 优化器、MSE 损失、Dropout 抑制过拟合

### （二）训练收敛性

训练集与验证集 Loss 同步呈负指数下降，无震荡 → 训练效果符合预期。

### （三）预测精度对比

| 评估指标 | XGBoost 模型 | LSTM 验证模型 |
|---------|---------------|----------------|
| R² | 0.6293 | **0.6619** |
| RMSE | 0.0051 | **0.0046** |

### （四）稳健性结论

1. **核心驱动因子高度一致**：灌溉率、洪涝占比、日照时数 Top 3 在两类模型中完全一致。
2. **作用方向逻辑自洽**：灌溉率高值集中于负半轴；洪涝占比高值集中于正半轴。
3. **特征梯度分布自然**：辅助因子（农机化、化肥）表现出预期的微弱调节作用。

## 六、生产韧性提升

### （一）风险特征三维度分析

1. **自然气候风险**：呈"北旱南涝、东西分化"异质性；气候异常 → 病虫害爆发 → 粮食减产链式传导；累积效应削弱系统自我修复能力。
2. **生产要素风险**：北方灌溉用水需求扩大、南方水资源浪费；农资价格随国际市场波动；劳动力外流。
3. **系统传导风险**：政策保障不完善；产业链上下游传导；主销区粮价受主产区风险冲击。

### （二）针对性建议（事前预防 - 事中应对 - 事后恢复）

1. **强化科技赋能**
   - 抗逆性品种研发与定向培育
   - 物联网+大数据农田监测系统
   - 智能化设备（无人机植保、精准灌溉）
   - 专业化技术队伍下乡

2. **完善基础设施**
   - 主产区物资储备库（化肥、农机）
   - 台风/强对流地区设施加固
   - 高标准农田建设、中小型水利设施

3. **健全制度保障**
   - 扩大农业保险覆盖面
   - 多部门协同风险防控机制
   - 市场宏观调控稳价

### （三）粮食生产风险可视化决策支持系统设计构想

**1. 三层架构**

| 层 | 内容 |
|----|------|
| 数据层 | 中国统计年鉴 + CMDC + 水旱灾害公报；自动化预处理（插值、标准化、滤波） |
| 模型层 | XGBoost-SHAP（归因）+ LSTM（时序预测），交叉验证 |
| 应用层 | Python Dash / ECharts / Leaflet 交互式前端 |

**2. 五大功能模块**

1. **风险地图**：省域 GIS 多维度切换
2. **因子追溯**：SHAP 全局/瀑布/交互热力图
3. **韧性评估**：韧性指数雷达图
4. **情景仿真**：滑动条调节因子，反事实预测
5. **预警推送**：分级预警 + 邮件/短信/API

**3. 应用价值**：政府决策、产业差异化、科研开放接口。

## 七、结论与展望

### （一）研究结论

**1. 核心因子排序**

灌溉率 > 洪涝占比 > 日照时数 > 水灾受灾面积 > 旱灾受灾面积 > 平均气温 > 农机总动力 > 干旱指数 > 降水量 > 化肥施用量

**2. 模型性能**

| 模型 | R² | RMSE | 特点 |
|------|------|------|------|
| XGBoost | 0.6293 | 0.0051 | 较好拟合，SHAP 强解释性 |
| LSTM | **0.6619** | **0.0046** | 时序稳健性更好 |

**3. 综合提升思路**：基础设施为基（灌溉、农机）+ 综合政策为支撑 + 品种技术为核心 + 可视化系统落地。

### （二）局限与展望

四方向改进：
1. 模型结构调整（交叉验证、注意力机制、GRU/Transformer、多模型集成）
2. 特征工程优化（去共线性、衍生特征、异常值处理）
3. 数据扩展（地市/县域微观、长周期）
4. 分区域建模 + 多气候情景仿真 + 可视化系统工程落地
