## 摘 要

在全球气候变化背景下，极端天气频发显著增加我国粮食生产系统风险，具有多因子联动及非线性特点。本文选取 2011—2023 年中国 31 个省份面板数据，**首次提出 XGBoost-SHAP + LSTM 双重验证的双层闭环研究思路**。

**实证结果：**
- XGBoost：R² = 0.6293，RMSE = 0.0051
- LSTM：R² = 0.6619，RMSE = 0.0046

**核心发现：** 灌溉率、洪涝面积占比和日照时长是主因素；灌溉率抵御风险，洪涝和光照变化加剧风险；非线性相互影响显著；双重检验下结论高度一致。

**关键词：** 极端气候；粮食生产风险；XGBoost-SHAP；LSTM；韧性提升

## 一、引言

### （一）研究背景与意义

- IPCC《Climate Change 2022》指出气候变化加剧农业脆弱性[1]。
- 2023 年台风"杜苏芮"致河北约 400~500 万吨玉米受灾（占全国 2%），夏粮产量减少 127 万吨。
- "十四五"规划与中央一号文件提出筑牢粮食安全根基[2]。

**研究空缺：** 单一气候因子线性研究居多，机器学习注重精度但忽视可解释性。

**本文路线：** 风险检测 → 机制解析 → 双重模型稳健性验证；XGBoost 非线性建模 + SHAP 拆分 + LSTM 时序验证 + 跨方法一致性检验。

### （二）研究内容

1. **核心因子识别**：粮食单产波动衡量风险大小。
2. **双模型构建**：XGBoost-SHAP（重要程度+相互影响）+ LSTM（时序稳定性）。
3. **韧性提升路径**：灾害防治、基础设施、政策保障。

### （三）创新点

1. **研究角度**：综合多种极端气候因素，由"被动防御"转向"主动预测"。
2. **模型方法**：XGBoost-SHAP 二元结构（高精度预测 + 透明归因）。
3. **应用价值**：可扩展的预警分析模型。

## 二、国内外研究现状综述

| 维度 | 国外 | 国内 |
|------|------|------|
| 研究脉络 | 单一气候因子 → 多极端事件耦合（低收入地区更显著）[7] | "风险评估—影响机理—预警防范"，区域性、季节性[8] |
| 评估方法 | 随机森林、XGBoost；深度学习时序 | 多用 XGBoost，单地区研究[9-12] |
| 韧性研究 | 抗旱品种 + 灌溉升级；多基于发达国家[13] | 起步晚但发展快；指标体系法；缺全链条研究[14-21] |

**三大不足：**
1. 单一气候因子或局部分析；
2. 静态韧性评估为主；
3. 单一模型应用，缺双重验证[3-5,22-27]。

## 三、数据预处理与建模方法

### （一）数据来源

- 农业生产：《中国统计年鉴》《中国农村统计年鉴》、各省统计公报
- 气象监测：CMDC 省域逐日数据
- 致灾因子：《中国水旱灾害公报》

### （二）三步预处理

1. **缺失值补全**：线性插值
2. **去趋势化降噪**：Butterworth 低通滤波 → 残差作为风险指标 Y → 二阶平滑
3. **Z-score 标准化**：均值 0、标准差 1

### （五）建模方法

> **注：** 本版本 docx 公式未正确转换，建议参照"融合优化版"获取完整公式。

1. **XGBoost**：加法集成 K 棵决策树 + 正则化项（叶子节点数 + 叶子权重 L2）
2. **SHAP**：基于 Shapley 值计算特征贡献度 → $\hat{y}_i = \varphi_0 + \sum \varphi_{ij}$
3. **LSTM**：遗忘门、输入门、输出门 + 细胞状态 + Dropout

### （六）评价指标：R²、RMSE

## 四、XGBoost-SHAP 核心因子识别

### （一）拟合性能

- 8:2 划分；网格搜索：lr=0.1，max_depth=6，n_estimators=150
- **测试集**：R² = 0.6293，RMSE = 0.0051

### （二）SHAP 全局重要性 Top 3

> **灌溉率 > 洪涝占比 > 日照时数**

### （三）蜂群图分析

- 韧性因子（灌溉率/日照）高值在 0 轴左侧 → 减损
- 致灾因子（洪涝占比/降水量）高值在右侧 → 推高风险
- SPEI 蓝色点（干旱）SHAP 为正 → 缺水冲击大

### （四）灌溉率 × 平均气温交互

- 主效应：灌溉率提升 → SHAP 下降
- 交互：高温会抵消灌溉减损效果

### （五）韧性提升方向

1. 强化水利设施 → 物理韧性
2. 多因子协同调控 → 系统抗性
3. 健全精准预警体系 → 管理韧性

## 五、LSTM 双重验证

### 模型架构

输入层 + 2 层 LSTM 隐藏层 + 全连接输出 + Adam + MSE + Dropout

### 性能对比

| 指标 | XGBoost | LSTM |
|------|---------|------|
| R² | 0.6293 | **0.6619** |
| RMSE | 0.0051 | **0.0046** |

### 稳健性

1. **核心因子高度一致**：Top 3 = 灌溉率、洪涝占比、日照时数（跨算法普适）
2. **作用方向自洽**
3. **特征梯度自然**

### 预测视角下的提升策略

1. 动态预警机制（LSTM 时序可预估，灾前预防）
2. 跨区域差异化防控（高波动地区强化基建；高频低波动地区健全保险）
3. 预测-识别-干预闭环（XGBoost-SHAP 定因 + LSTM 预估 + 精准干预）

## 六、结论与展望

### （一）结论

**1. 因子排序：** 灌溉率 > 洪涝占比 > 日照时数 > 水灾面积 > 旱灾面积 > 平均气温 > 农机 > 干旱指数 > 降水量 > 化肥施用量

**2. 模型性能：**
- LSTM：R² ≈ 0.66，残差近正态分布、无自相关、时序稳健
- XGBoost：R² = 0.6293，结构合理

**3. 综合策略：** 基础设施为基础 + 综合政策为支撑 + 品种技术为核心 → 多维度协同

### （二）局限与展望

四方向改进：
1. 加交叉验证、注意力机制、GRU/Transformer、多模型集成
2. 去共线性、衍生特征、异常值处理
3. 拓展数据周期、地市/县域微观数据
4. 分区域建模、多气候情景仿真、预警系统落地
