| 项 | 阿里云 | 腾讯云 | Namesilo |
|---|---|---|---|
| .cn 价格（首年） | ~29 元 | ~29 元 | 不卖 .cn |

| 是否可在国内备案 | ✅ | ✅ | ❌ 域名不在境内服务商，不能走国内 ICP 备案 |
| 学生优惠 | ✅ 阿里云高校计划 | ✅ 腾讯云高校计划 | — |

**推荐**：阿里云 `.cn` 域名 + 阿里云轻量应用服务器（学生优惠 9.5 元/月起），全套打通。

### 1.2 域名候选

需要在去注册前先确定 3-5 个候选名（被占的概率高）。命名建议：

- 主题相关 + 短：`grainrisk.cn` / `cropshield.cn` / `agririsk.cn` / `chinacropai.cn`
- 学校/项目缩写 + 主题：`sxauagri.cn` / `dc-grain.cn`（DC = 大创）

> **行动项**：潘妙齐本人在阿里云域名查询页 https://wanwang.aliyun.com/domain/ 实际查询 5 个候选，挑一个未被占用的**今日**注册下来。

### 1.3 实名认证（注册当天要做）

- `.cn` 域名要求在 5 个工作日内通过实名认证，否则 hold
- 上传：身份证正反面 + 持证人脸照
- 模板：阿里云控制台 → 域名 → 实名认证 → 选 "个人" → 提交

## 3. ICP 备案（**关键路径**）

### 3.1 备案主体

- 类型：**个人备案**（潘妙齐本人，山西农业大学学生身份）
- 用途：非经营性（项目演示/学术展示，不卖东西、不收会员费）
- 备案省份：**山西省**（与身份证地址一致）

### 3.2 材料清单（一次过，避免被驳回返工）

| 材料 | 来源 | 说明 |
|---|---|---|
| 身份证正反面 | 本人 | 高清扫描或照片，四角完整 |
| 工信部公安联网核查电子化人像 | 阿里云 App | 在阿里云 ICP 代备案管理系统 App 内活体认证（手机操作） |
| 域名证书 | 阿里云控制台 | 域名 → 域名信息 → 下载域名证书 |
| 服务器 IP / 备案码 | 阿里云控制台 | 轻量服务器购买后自动生成 |
| 网站名称 | — | 不能含"中国"/"国家"/"大学"等敏感词；建议**项目名缩写 + 描述**，例如"极端气候粮食生产风险分析系统" |
| 网站负责人电话 + 邮箱 | 本人 | 用 `15934110986pmq@gmail.com` + 本人手机号 |
| 应急电话 | 本人 | 与负责人电话不同的备用号（可填导师邱东芳老师的，需她同意） |

### 3.3 山西省管局补充要求（容易卡）

- 网站名称含"管理系统"/"平台"等可能被要求**提供前置审批**——避雷起见**不要写**这些词，写"展示系统"、"分析系统"是安全的。
- 个人备案**不允许**网站内容包含论坛、博客互动、用户注册登录登记等"经营性"特征——但本项目是**单向展示**（地图+SHAP 看板+情景模拟+路径推荐），都是只读，**符合个人备案范围**。
- 个人备案如果含"用户注册/JWT 鉴权"，山西管局有概率驳回——本项目 JWT（doc 00 § 4.2 任务 3.3）只用于**项目组成员后台调试**，不对公众开放。备案前应把对外页面的"登录入口"做成**隐藏路由**或部署到子域名（例如 `admin.<domain>`）单独管理；备案信息里只写主站内容。

### 3.4 提交流程

1. 阿里云 ICP 代备案管理系统 https://beian.aliyun.com/ → 新增备案 → 选择服务器
2. 填写主体信息（个人）→ 网站信息（域名 + 名称 + 负责人）
3. App 端活体认证（拍证件 + 人脸）
4. 阿里云**初审 1-3 个工作日**
5. 通过 → 提交至**山西省通信管理局**复审
6. 管局**审核 1-20 个工作日**（山西省过往实测中位数 ~12 天）
7. 备案号下达 → 域名可对外解析 80/443

### 3.5 备案期间需要做的事

- 服务器**必须保持开机**，nginx 跑一个最小占位页（写"网站正在建设中"足够）
- 不要部署任何对外功能（可访问的 API 都要先关）
- 域名解析**先指向阿里云**默认 IP（备案模板页），不要指向自己服务器
- 备案号下来后**5 个工作日内**必须把备案号挂到网站底部（管局会扫描，没挂会被注销）
