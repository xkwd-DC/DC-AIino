# 业务理解

- 软件名称：极端气候下粮食生产风险核心因子识别与韧性提升可视化决策系统
- 产品定位：极端气候下粮食生产风险核心因子识别与韧性提升可视化决策系统是一套面向省级粮食生产管理的智能分析平台，集成多模态气候-统计-遥感数据，通过XGBoost-SHAP、基线LSTM、Attention-LSTM三模型互补归因，识别影响粮食单产的关键风险因子，并为决策者提供可交互的情景推演与韧性路径建议。
- 面向领域 / 行业：农业信息化 / 粮食安全决策支持
- 核心价值：将31省13年(2011-2023)的11维多模态数据与三套机器学习模型融合，诚实披露三协议鲁棒性边界(random 8:2 R²=0.907、leave-year-out R²=0.894、leave-province-out中位-16.83)，在省内可解释归因基础上提供参数情景模拟与韧性路径推荐，帮助用户在已知模型局限下做出有依据的农业风险管理决策。
- 证据文件：`业务理解证据.md`

## 目标用户

- 省级农业管理部门及政策制定人员
- 农业科研机构研究人员
- 高校农业政策研究者
- 政府涉农决策参谋人员

## 主要业务功能

- 系统总览：展示数据规模（31省、403样本、11维特征）和三模型R²指标，呈现数据融合进度及三协议鲁棒性披露
- 风险时空地图：以交互式中国地图展示各省年度风险等级着色，支持年份滑块切换历史数据，点击省份查看历史产量曲线
- SHAP归因看板：对选定省份运行XGBoost-SHAP归因，以11维特征归一化重要性柱状图展示各气候与管理因子的边际贡献方向和大小
- 参数情景模拟：用户调节灌溉率、洪涝占比、日照时数、平均气温、SPEI干旱指数5维滑块，前端公式实时反馈风险指数变化，同步调用后端三模型异步确认
- 韧性路径推荐：依据省份现状与全国均值偏差，由11条规则引擎生成短期（1年）、中期（3年）、长期（5年）行动建议，估算介入预算
- 推演监控：每3秒轮询后端，实时展示M02/M03/M04触发的三模型协同推演日志，记录输入参数、各模型输出、集成结果及响应时延

## 功能说明

- 系统总览：展示数据规模（31省、403样本、11维特征）和三模型R²指标，呈现数据融合进度及三协议鲁棒性披露：进入系统默认落点页面，以4张KPI卡展示覆盖省份(31)、数据样本(403)、三模型random 8:2 R²(0.907)和leave-province-out中位R²(-16.83)。下方呈现XGB-Top3(NDVI/降水量/气温)与Attention-LSTM-Top3(旱灾面积/SPEI/气温)的互补归因对比，以及数据融合进度条(省级统计100%/MODIS遥感100%/CLCD耕地掩膜60%)。
- 风险时空地图：以交互式中国地图展示各省年度风险等级着色，支持年份滑块切换历史数据，点击省份查看历史产量曲线：主区域为ECharts渲染的中国省级GeoJSON地图，色阶按风险分位数着色。顶部年份滑块(2011-2023)控制当年数据，切换年份时向/api/provinces?year=发起请求并缓存结果。点击省份后右侧展示该省历史产量折线图及当年11维特征数值卡片。
- SHAP归因看板：对选定省份运行XGBoost-SHAP归因，以11维特征归一化重要性柱状图展示各气候与管理因子的边际贡献方向和大小：左侧省份选择器与状态共享。调用后端接口返回11维归一化重要性。主区域为水平柱状图，每维特征附方向色(绿=有利/红=有害)。辅助区展示全局归因子集对比，可切换农业条件/气候因子/灾害因子三类视角。
- 参数情景模拟：用户调节灌溉率、洪涝占比、日照时数、平均气温、SPEI干旱指数5维滑块，前端公式实时反馈风险指数变化，同步调用后端三模型异步确认：5条滑块(灌溉率/洪涝占比/日照时数/平均气温/SPEI)以选中省份基线值为起点，拖动后前端公式立即更新风险指数(200ms防抖)；同时向后端发异步请求，三模型集成结果覆盖前端估算。ECharts雷达图对比调参前后11维特征偏移。
- 韧性路径推荐：依据省份现状与全国均值偏差，由11条规则引擎生成短期（1年）、中期（3年）、长期（5年）行动建议，估算介入预算：页面左侧展示选中省份与全国均值对比的3项指标(风险指数Y/灌溉率/洪涝占比)。中心区为规则引擎生成的短/中/长期行动卡片，每张卡包含行动描述、优先级标签和预估介入效果。页面底部估算总预算范围。
- 推演监控：每3秒轮询后端，实时展示M02/M03/M04触发的三模型协同推演日志，记录输入参数、各模型输出、集成结果及响应时延：每3秒轮询后端，展示最近50条推演记录表格。每行含触发时间戳、输入省份、三模型预测值(XGBoost/LSTM/集成)、响应时延。点击行展开11维输入参数明细。顶部展示平均响应时延和缓冲区占用率。

## 典型操作流程

1. 打开系统后进入系统总览(M00)，浏览数据规模KPI卡和三协议鲁棒性边界指标
2. 切换到风险时空地图(M01)，拖动年份滑块选取目标年份，点击目标省份查看历史产量走势
3. 切换至SHAP归因看板(M02)，选择省份，查看11维因子重要性排序和归因视角
4. 进入参数情景模拟(M03)，调整灌溉率、气温等滑块，观察风险指数实时变化
5. 切换到韧性路径推荐(M04)，查阅短期、中期、长期行动建议及预算估算
6. 如需查看历史推演记录，进入推演监控(M05)，查看三模型协同推演日志和响应时延

## 操作手册页面/流程模块

1. 系统总览：打开系统网址后自动跳转至本页面，或在顶部导航点击「M00 系统总览」。
   - 使用场景：用户刚打开系统时查看整体数据规模和模型可信度边界，或向同事演示系统能力时使用本页面。
   - 操作要点：浏览页面顶部4张KPI卡，了解数据规模和三模型指标；查看归因对比区，了解XGB与Attention-LSTM在Top3因子上的异同；查看数据融合进度条，了解当前数据集各来源的就绪状态；点击页面中的模块入口卡片，或使用顶部导航切换到其他功能模块
2. 风险时空地图：点击顶部导航「M01 风险时空地图」进入。
   - 使用场景：农业研究人员分析某一历史年份全国粮食风险格局，或需要了解特定省份近年产量趋势和当年气候条件组合。
   - 操作要点：进入页面后地图自动加载最新年份数据，各省按风险着色；拖动年份滑块切换历史年份，地图着色随之更新；将鼠标悬停在省份上，tooltip显示该省名称和当年风险指数；点击某一省份，右侧面板展示该省近年历史产量折线图及当年11维特征指标
3. SHAP归因看板：点击顶部导航「M02 SHAP归因看板」进入，或从系统总览的模块入口卡进入。
   - 使用场景：研究人员需要了解某省粮食风险由哪些因子主导，或需要对比不同省份在因子重要性上的差异。
   - 操作要点：进入页面后，左侧或下拉框选择目标省份；系统自动向后端发起预测请求，返回11维SHAP归一化重要性；主区域展示11维水平柱状图，按重要性降序排列；点击视角切换标签（农业条件/气候因子/灾害因子）缩小维度范围查看
4. 参数情景模拟：点击顶部导航「M03 参数情景模拟」进入。
   - 使用场景：决策参谋评估灌溉率提升或气温升高等干预措施对风险指数的影响，或研究人员系统比较多个参数组合下的风险响应。
   - 操作要点：进入页面后选择目标省份，滑块自动以该省基线数据初始化；拖动任一滑块调节参数值，前端风险指数在200ms内更新估算值；约0.5秒后后端三模型集成预测结果覆盖前端估算值；查看雷达图了解当前参数与基线的11维偏差分布
5. 韧性路径推荐：点击顶部导航「M04 韧性路径推荐」进入，或从系统总览的模块入口卡进入。
   - 使用场景：农业主管部门为某省制定近期和中长期农业韧性提升计划，希望系统给出基于数据的针对性行动建议和预算参考。
   - 操作要点：进入页面后选择目标省份；左侧指标卡展示该省风险指数、灌溉率、洪涝占比与全国均值偏差；向下滚动查看短期、中期、长期行动建议卡，了解各阶段推荐措施和优先级；查看底部总预算估算，结合实际情况参考规划
6. 推演监控：点击顶部导航「M05 推演监控」进入。
   - 使用场景：研究人员回溯某次情景模拟的具体输入参数和三模型输出差异，或监控后端推理性能和服务响应时延。
   - 操作要点：进入页面后系统自动开始每3秒刷新推演记录；浏览表格中各行的时间、省份和三模型预测结果；点击某行展开，查看该次推演的11维完整输入参数；关注顶部平均时延指标，了解当前系统响应性能

## 申请表建议口径

- 开发目的：为省级农业风险管理提供可解释、可推演的粮食生产风险分析工具，支撑极端气候背景下的韧性农业决策
- 软件的主要功能：本软件主要功能包括以下六个方面。一、系统总览：汇聚31省403样本的多模态数据集关键指标，以KPI卡形式展示三模型在三种验证协议下的R²分布，并对比XGBoost-SHAP与Attention-LSTM归因的Top3因子差异，帮助用户在使用各模块前建立对数据规模和模型可信度的整体认知。二、风险时空地图：以省级GeoJSON交互地图呈现2011至2023年各省粮食生产风险等级的时空分布，用户可拖动年份滑块切换历史数据，点击省份后查看该省粮食单产历史折线图及当年11维特征指标，直观把握区域风险格局演变趋势。三、SHAP归因看板：针对用户选定省份，调用后端XGBoost-SHAP归因接口，以归一化重要性水平柱状图展示11维气候、致灾与管理特征各自的边际贡献方向和量级，支持在全局、农业条件、气候因子、灾害因子四个视角间切换，帮助用户识别当前省份粮食风险的主控因子。四、参数情景模拟：提供灌溉率、洪涝占比、日照时数、平均气温、SPEI干旱指数5维可调参数滑块，调节时前端线性公式实时反馈风险指数估算，同步向后端发起三模型集成预测请求，以雷达图叠加展示调参前后11维特征偏移，支持用户快速评估不同干预措施对风险的影响方向和量级。五、韧性路径推荐：依据选定省份在风险指数、灌溉率、洪涝占比等指标上相对全国均值的偏差，由内置11条规则引擎自动生成短期（约1年）、中期（约3年）、长期（约5年）三档行动建议卡，每张建议卡包含具体措施描述、优先级评级及预估介入效果，并综合计算总预算区间供规划参考。六、推演监控：实时轮询后端推演历史接口，以时间倒序列表展示近50条查询记录，每条记录包含触发时间、输入省份、XGBoost与LSTM的单独预测值及三模型集成结果、接口响应时延，支持展开查看完整11维输入参数，帮助用户追踪和复核历次风险分析的完整输入输出过程。
- 技术特点：Vue 3+TypeScript前端与Flask 3 Python后端分离架构，集成XGBoost、LSTM、Attention-LSTM三模型互补归因，ECharts交互式数据可视化，Pinia状态共享

- 软件分类：应用软件
